Array definition file

Is a file with mappings of columns/rows to an ORF

Control borders

is the borders around the plate which are controls, which are ignored throughout the analysis. For example, a control border of 2 will ignore the first 2 and last 2 column/rows on the plate

Array replicates

This value is complementary to the array definition file (if provided). If the plate has replicates, this value specifies how the replicates are ordered. A value of 4 indicates that replicates of arrays are in quadruples.
Rows and columns can then be assigned to replicates which, in turn, are used to map ORFs from the array definition file, if provided. If not provided, the rows and columns generated serve as the ORFs

Linkage correction

If selected, ignores genes from the analysis, which are on the same chromosome and are within a specific proximity (as specified by the user in KB) to one another

Score results

This option allows you to score your normalized plates given control plates or array single mutant fitnesses is supplemented.
